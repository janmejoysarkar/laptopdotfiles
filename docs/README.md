# Desktop Dotfiles
Dotfiles for `janmejoy@machine`

# What's new?
- Switched to `sway` window manager. `sway` runs on Wayland.

# Dependencies for `sway`
- `awww`- Wallpaper manager
```
awww img path/to/file.png
```
- `grim` for screenshots 
- `slurp` for specifying screenshot region. To be piped into `grim`
- `waybar`
- `swaylock`
